# range of repetitions in an array  
```jsbp
[Int{2, 3}]
```

## empty array
```json
[]
```
+ Invalid
    - `.`

## singleton array
```json
[0]
```
+ Invalid
    - `.`

## 2 element array
```json
[0, 1]
```
+ Valid

## 3 element array
```json
[0, 1, 2]
```
+ Valid

## 4 element array
```json
[0, 1, 2, 3]
```
+ Invalid
    - `.[3]`
